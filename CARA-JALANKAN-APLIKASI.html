Cara Menjalankan Aplikasi
Aplikasi belum berjalan! Silakan jalankan terlebih dahulu.
1 Jalankan Semua Aplikasi (MUDAH)
Double-click file berikut untuk menjalankan semua aplikasi sekaligus:
start-all-apps.bat
File ini akan otomatis membuka 5 terminal dan menjalankan semua aplikasi.
2 Tunggu Aplikasi Siap
Tunggu hingga semua terminal menampilkan pesan "ready" atau "running". Biasanya membutuhkan waktu 10-30 detik.
3 Buka Aplikasi
Setelah semua aplikasi berjalan, buka index.html di browser atau klik tombol di bawah.
Cara Manual (Jika Otomatis Gagal)
Jika start-all-apps.bat tidak bekerja, jalankan manual satu per satu:
  1. Buka terminal, jalankan: cd backend && npm run dev
  2. Buka terminal baru, jalankan: cd registration-app && npm run dev
  3. Buka terminal baru, jalankan: cd operator-app && npm run dev
  4. Buka terminal baru, jalankan: cd display-app && npm run dev
  5. Buka terminal baru, jalankan: cd petugas-app && npm run dev
