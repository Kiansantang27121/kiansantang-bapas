## ğŸ› Troubleshooting

### Preview tidak muncul

**Check:**
- Button Preview diklik
- No JavaScript errors
- Browser support modal

**Solution:**
- Refresh page
- Check console

### Preview layout berbeda

- Settings tersimpan
- Cache browser
- Component updated

- Clear cache
- Hard refresh
- Check settings state

- Theme settings filled

- Settings passed to preview

- Check color picker
- Verify hex values
- Re-select colors

### Text tidak berganti

**Check:**
- Multiple texts filled
- Text not empty
- Rotation interval

**Solution:**
- Fill at least 2 texts
- Wait 3 seconds
- Check console errors
